Frequently Asked Questions

What image formats are supported?
We support JPEG and PNG image formats. For best results, use high-quality images with clear visibility of the cattle. The maximum file size is 10MB.
How accurate is the breed identification?
Our AI model achieves 92% accuracy on validation data. Accuracy can vary based on image quality, lighting conditions, and cattle positioning. Including additional trait data improves prediction accuracy.
What cattle breeds are currently supported?
We currently support 5 major breeds: Gir (Indigenous Indian), Holstein-Friesian (H_F), Jersey, Murrah (Buffalo), and Nili Ravi (Buffalo). Additional breeds are being added based on user demand.
How long does processing take?
Image processing and breed prediction typically takes 200-500ms depending on image size and server load. Results are delivered in real-time.
Can the system detect crossbreeds?
Yes, our system includes crossbreed detection. When confidence scores are low or multiple breeds show similar characteristics, the system flags potential crossbreeds.
Is there an API rate limit?
Development usage allows 100 requests per hour. Production API keys have higher limits. Enterprise customers receive custom rate limits based on their needs.
How can I improve prediction accuracy?
For best results: use high-quality, well-lit images; include the full body of the cattle; provide additional trait data (milk production, weight, etc.); ensure the cattle is clearly visible without obstructions.
